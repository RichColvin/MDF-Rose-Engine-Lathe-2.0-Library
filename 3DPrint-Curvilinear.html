1001
Finials Template

The files are for all 3 pieces. The G-Code file is set to print all 3 in one printing.

1001-A & 1001-B have a groove in the design to allow them to be aligned whenever stacked. Radii are shown in the picture to the left.

1001-C is a spacer used below whichever one is stacked above the other (its use is not always required).

All parts are 1/8" thick. 1001-A & 1001-B are 7" long.
