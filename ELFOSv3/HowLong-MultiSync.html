How Long Will a Synchronized Pass Take?

This calculator will give you an estimate of how long a MultiSync synchronized movement should take.

Spindle Speed
Max Pulses per Sec   This is set at the top of the speed slider for the spindle.
Speed Percentage  % This is the percentage of the maximum speed for the spindle. It is set on the speed slider for the spindle.
Movement Distance
Linear Distance Moved  in The linear distance moved (e.g., X, Z, or M4, assuming that M3 is dedicated to the rosette phaser/multiplier). Typically, this is Z.
Spindle Degrees  ° The degrees the spindle is rotated.
Configuration Values
Driver Pulses / Rev  pulses

This is the number of microsteps your driver is set to use per revolution of the stepper motor.

6400 is the default for ELFOS on the MDF Rose Engine 2.0.

Motor Rev / Spindle Rev  

This is the number of revolutions of the stepper motor is required to revolve the spindle 1 complete turn.

9 is the standard value for ELFOS on the MDF Rose Engine 2.0.
