Preferences Screen
Load ini
Purpose: Used to load settings for the system from a file. The text fields shown are just to show that the command is working.

Notes:

  1. You can load a file which only has partial settings, and not all of them. This is useful for saving the settings for devices (e.g., a file for the curvilinear slide, and a different file for a spherical slide).
  2. You can also load a settings file which isused for a specific device. For example, the settings for a curvilinear slide could be loaded with no other ones affected.
Save Settings
Purpose: Used to save the current system settings to a file named Settings.ini.

Notes:

  1. When using this action, any file named Settings.ini will first be erased, and then a new one created to save the current system's settings.
