Signal Light

Auxiliary Pin This is the pin number used for identifying when an operation is running.

When using certain functions (such as Sync and MultiSync), the final pass can take quite a long time -- many hours is not uncommon. This function was requested so that a signal light can indicate when the activity is still underway.

The idea was to have a light (such as the one to the right) lit when activity is underway, and not lit when the activity has stopped. This allows the operator to walk away from the machine when using these long-running operations, and monitor the activity periodically by glancing over to the machine to identify if the activity has stopped.

More informataion about this is at Auxiliary Pin Function.

Keep Steppers Enabled

This is used to set the stepper motors to be continuously Disabled or Enabled.

  • If you are using DM542T stepper motor drivers, this needs to be Enabled for any operations where repeated movement relies on the current position being correct. One example is indexing. If this is not set for indexing, the movement will be off with each individual index.

    It is recommended that the user

    • Keep Steppers Enabled &
    • Keep M3 Enabled (if using a rosette phaser/multipler).
  • If you are using the DRV8825 stepper motor drivers, it can be set either way.
