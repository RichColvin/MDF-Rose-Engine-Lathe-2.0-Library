Preferences Screen

Touch the buttons on the screen image to the right to see the details for the that screen.
Purpose: Used to load the INI file. The text fields shown are just to show that the command is working. These boxes only appear when Load Ini has been touched.

The option introduced with version 3.0 is to select the file to load.. This allows for a settings file to be used for a specific device. For example, the settings for a curvilinear slide could be loaded with no other ones affected.

  2. Limits
  3. Returns
  4. Main
  5. Multi
  6. Index
  7. Move
