ELFOS
Limit Switches and the EStop Function

Normally Open Switches

If you want to have more than one switch for the quick stop function, and you have configured your system to use switches which are normally open, you need to wire them in parallel.

This way, if any of the switches is depressed, the selected pin # is shorted to GND.

Normally Closed Switches

If you want to have more than one switch for the quick stop function, and you have configured your system to use switches which are normally closed, you need to wire them in series.

This way, if any of the switches is depressed, the pin # is no longer shorted to GND.
