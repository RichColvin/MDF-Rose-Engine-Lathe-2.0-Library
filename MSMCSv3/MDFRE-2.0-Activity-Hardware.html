Teensy Controller &
Nextion Touch Screen Display

The computing hardware that can be used is outlined below, along with a discussion about the benefits of each.

Teensy Controller
Model Commentary
3.2
  • These are somewhat available these days. Finding a Teensy 3.5 or 3.6 is very hard with all the chip shortages these days.

  • This controller does not have an on-board microSD card reader. An external one like this Micro SD card reader module for Arduino from tindie is needed. The Teensy 3.2 requires an adapter which handles 5 VDC. Many other adapters are 3.3 VDC.

  • This controller can only handle 4 stepper motor drives.

  • This controller runs at 72 MHz, 40% slower than the Teensy 3.5. However, this slower speed is not restrictive.

TEENSY32 on PJRC's web page.

3.5
  • Most commonly used -- when they are available.

  • This controller can only handle inputs at 3.3 VDC or 5 VDC.

  • This controller can only handle 3.3 VDC. Be sure to not overload it with 5 VDC on any inputs.

TEENSY35 on PJRC's web page.

3.6
  • This controller runs at 180 MHz, 50% faster than the Teensy 3.5. However, this extra speed is not needed.

  • This controller can only handle inputs of 3.3 VDC. Be sure to not overload it with 5 VDC on any inputs.

TEENSY36 on PJRC's web page.

Other models of the Teensy are not supported at this time.

Nextion Display
Model Commentary
4.3" Enhanced
  • Most commonly used

NX4827K043 on Nextion's web page.

7" Intelligent
  • More and more used.

  • The intelligent displays operate at 5 VDC. It could overload a Teensy 3.6.

NX8048P070-011R on Nextion's web page.

Other models of the Nextion display are not supported at this time.
