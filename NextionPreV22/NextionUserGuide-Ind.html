Index Screen
Divisions Option

Purpose: This screen allows indexing the spindle for use in index work or other uses.

1 , 2 , 3 , 4 , and 5 may each represent separate saved settings (aka, pre-sets), however the value of 1 is used by the CW and CCW indexing movements on the

  • Synchronization Screen and the
  • Reciprocation - Axis Screen
in either Divisions or Degrees, depending upon which is toggled.

Having 5 pre-sets allows the ornamental turner to easily repeat a prior setup.

On this screen, the indexes are in Divisions of a circle, however each of the 5 pre-sets can be in divisions of a circle or in degrees. The selection is separate for each pre-set.

Deg
Touch Deg on the screen image to see the instructions for that approach.
Sets the divisions to degrees.
 
Div Used to set the indexing to divisions of a circle.
File
Touch the Filename field (shown as "INDEX_02") to see the details for using a file for indexing.
Used to set the indexing to information to be fed from a user-created file.
 
Fixed Sets the increments based on the Divisions setting.
  Divisions   Sets the amount of each index as a fraction of a circle. \[ Indexing \, \mathrm{deg} = \frac{360 \, \mathrm{deg}}{Divisions} \]
  Index   This is the number of times the axis has been indexed. It is decremented each time the spindle is moved counter clockwise, and incremented when moved clockwise.

Tapping on this does enable the user to reset the index; however that does not affect the position of the spindle.

Limit Switches

Limit switches cannot be used with this function. They can only be used from the Main Screen or One Screen.
